*************************************************************************
Release 4.5    Release Date: 23.04.2004
*************************************************************************

Начиная с версии 4.5 используется новый скрипт установки Build.PL. 

Добавлен новый вид ресурса в Каталоге: возможность добавления большого 
количества "статей" с группировкой по дате и еще чему-нибудь.

Внимание! Изменился алгоритм "автоформата" для текстовых ресурсов. Изменение 
может затронуть текстовые ресурсы Каталога и тексты сообщений в приложении 
MsgForum! 

Обязательно сделайте резервную копию вашей базы ePortal и рабочий вариант 
ePortal перед обновлением!

 

ePortal:
+ параметр конфигурации ePortal_dbi_source заменен на ePortal_dbi_host, ePortal_dbi_database
+ все приложения используют одино подключение к MySQL серверу. Если использовались
  разные источники dbi для приложений - эти базы надо скопировать в основную базу ePortal
- Новый вид ресурса в Каталоге
- для наиболее часто используемых типов файлов используется своя таблица MIME типов
- скрытые группы и ресурсы в каталоге
- настрока: показывать или нет присоединенные файлы ресурса
- информация о дате создания, последнем изменении ресурса
- в диалогах не сохранялись скроки в "кавычках"
- отпала необходимость в переменной PerlSetVar MasonEscapeFlags в httpd.conf
- немного расширены возможности автоформатирования ресурсов в Каталоге и сообщений в форумах
- изменен скрипт установки. Build.PL вместо Makefile.PL

SquidAcnt:
- не удалялись данные старее 2 месяцев

Release 4.4    Release Date: 30.01.2004
*************************************************************************

Исправлено несколько мелких ошибок.

ePortal:
- реестр изменений теперь и на русском языке
- изменилась структура БД. Обновление будет выполнено автоматически.
+ Изменился формат хранения домашних страниц. Надо перепроверить настройку
  ВСЕХ секций домашней страницы.
- проверка на существование такого пользователя при ручной регистрации
- из-за большого объема информации некоторые разделы при редактировании
  элементов каталога сделаны сворачиваемыми.
- возможность скрыть изображение <HTM>, обозначающее вставку HTML кода на странице
- ошибка Cannot open dir /usr/lib/perl5/site_perl/5.8.0/ePortal/ for reading 
  если установлен "чистый" ePortal без приложений. Исправлено.
- не работала кнопка "Применить" в некоторых диалогах
- не работал переход по страницам при большом количестве прикрепленных файлов
  в ресурсе каталога.

SquidAcnt:
- если любой из лимитов не определен, то никакой другой не 
  срабатывает. Исправлено.
- Недели при учете трафика считались по "американски", т.е. воскресенье-начало недели.

Removed:
- Удалено приложение compInventory (не закончено и не развивается)

WARNING!!! I have no news from English-speaking users of ePortal. 
If there is no such interest then I will finish that hard dual-language

OffPhones:

- export a department with subdepartments to text file 
- print preview for department with subdepartments (export to html)
