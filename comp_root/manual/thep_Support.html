NAME

ePortal::ThePersistent::Support - Helper package between ThePersistent classes and ePortal.

SYNOPSIS

ePortal::ThePersistent classes are entirely independent of ePortal itself. ePortal::ThePersistent::Support is medium layer between them.

METHODS

initialize()

Parameters are the same as ThePersistent::Base::initialize(). Additional parameters are:

  • DBISource
    This is DBISource passed to $ePortal-DBConnect>.
