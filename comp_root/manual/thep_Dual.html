NAME

ePortal::ThePersistent::Dual - Single row ThePersistent object.

SYNOPSIS

See ePortal::ThePersistent::Support and its base classes for more information. The Dual provides ``single record'' virtual storage.

It can be used to create pseudo persistent objects with use all power of Support.pm
