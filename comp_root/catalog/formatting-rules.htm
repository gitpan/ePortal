Some formatting rules

Paragraphs are separated by a blank line.

The horizontal lines are made with 4 or more dashes:
----

Text formatting
*bold text* /italic text/

= header 1=

== header 2==

=== header 3===

==== header 4====

Link to attached file: [filename.ext] or [filename.ext description of the file]

Link to a Catalogue resource named with nickname: [nickname] or [nickname description]

Link to anywhere: [http://some.server/ description of the link] <&| /rus.mc &>

Оформление текстов

При создании ресурсов Каталога, при написании сообщений на дискуссионных форумах, а также в некоторых других случаях можно использовать упрощенную систему для оформления итогового текста.

Для отделения одного параграфа от другого используйте пустую строку. 

 первый параграф
 вторая строка первого параграфа

 другой параграф
 и его вторая строка

Строки, которые начинаются с пробела выводятся без форматирования (см. пример чуть выше).

Форматирование текста осуществляется так:
*Жирный*, /Наклонный/, _ Подчерк _

[имя_файла.ext] [имя_файла.ext описание файла]
ссылка на прикрепленный файл (для ресурсов Каталога)

[короткое_имя описание ресурса]
ссылка на ресурс Каталога. Для этого надо указать его "короткое имя" и, возможно, свое описание

[http://www.server.com описание ресурса]
произвольная ссылка. http: или ftp:, mailto: - писать обязательно.

[image=/path/image.png]
Вставка картинки

[color=red some text] [style=font-weight:bold; this is styled text]
Вывод "some text" другим цветом или стилем. Пробелы в описании цвета или стиля не допускаются.

Оформление стилем "Заголовок" осуществляется соответствующим количеством знаков = в начале и конце строки:

= заголовок 1=

== заголовок 2==

=== заголовок 3===

==== заголовок 4====

Горизонтальная линия создается с помощью 4-х или более знаков -
